網頁設計 - HTML 外掛 Emmet

1. 認識 Emmet

Pug是一套文字編輯器的外掛程式，提供快速開發 HTML、CSS的程式碼格式。

2. HTML 語法

2-1 標籤

語法：標籤名稱 + Tab

2-2 標籤類別與識別

語法：標籤名稱 + 類別名稱 + Tab

語法：標籤名稱 + 識別名稱 + Tab

注意：如果不寫標籤名稱預設為div標籤

2-3 子標籤

語法：標籤名稱 + > + 標籤名稱 + Tab

2-4 乘法

語法：標籤名稱 + * 數字 + Tab

2-5 加法 - 兄弟姊妹

語法：標籤名稱 + + + 標籤名稱 + Tab

2-6 往上爬

語法：標籤名稱 + ^ + 標籤名稱 + Tab

兩個 ^ 等於往上爬兩次

2-7 群組

語法：( 標籤群組 ) + Tab

2-8 文字

語法：標籤名稱 + { 文字 } + Tab

2-9 屬性

語法：標籤名稱 + [屬性名稱=屬性值] + Tab

2-10 數字

數字，可加在標籤、類別、識別與文字上

語法：$ + Tab

指定數字從哪開始

語法：$@數字 + Tab

顛倒

語法：$@-數字 + Tab

2-11 省略標籤名稱

2-2 介紹：標籤名稱 + .類別名稱 如果標籤名稱省略則預設為 div，但在不同標籤內則有不同預設標籤

語法：標籤名稱 > .類別名稱 + Tab

也適用於識別名稱

3. CSS 語法

3-1 數值

語法：屬性 + 數值縮寫 + Tab

百分比p

eme

3-2 顏色

語法：屬性 + 顏色縮寫 + Tab

3-3 加法

語法：屬性 + 值 + + + 屬性 + 值 + Tab

3-4 縮寫

大部分為屬性的第一個字母，例如：顏色為c

如果遇到有連字號的為兩個單字的第一個字母，例如：背景顏色為bc

有一些特例，因為屬性有許多很相似的單字，例如：字型尺寸為fz
