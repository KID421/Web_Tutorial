網頁設計 - HTML 基礎

1. 認識表單

表單應用可以說是互動網頁不可或缺的部分，不管是任何型態的網站都可能會用到表單，最常應用於註冊、登入與聯絡頁面。

2. 表單語法

<form
表單元素
action="XXX.php"
表單資料傳送頁面
method="get">
資料傳送方式

注意：所有表單元素都必須放在<form></form>內。

2-1 資料傳送方式

GET POST
網址 輸入的資料會顯示在網址上，網址會改變 網址不會改變
傳遞量 由於是透過網址傳遞，所以有長度限制 沒有長度限制
安全性 內容會顯示在網址上，所以不安全 內容不會顯示在網址上，所以比較安全

可以到W3School GET頁面測試。

3. 表單元素

表單有許多不同的欄位，可以參考W3School裡面的表單元素列表。

3-1 輸入元素

輸入元素有多種不同的類型，例如：文字、信箱或密碼等等..

在W3School裡面的type列出了所有的輸入類型。

<input type="text" >
<input type="password" >

表單送出後會由後台讀取資料，所以每個表單元素都必須搭配一個name讓後台知道資料名稱。

<input type="text" name="user" >

如果此欄位為必須填寫並且需要檢查，可以加入required讓表單送出時檢查此欄位。

<input type="text" name="user" required >

3-2 表單其他元素

參考W3School裡面的表單元素列表並實作表單頁面。
