網頁設計 - HTML 入門

1. 認識 HTML

HTML 為一種【標記語言】，透過元素來告知瀏覽器要顯示的內容。

2. 認識元素

元素由標籤與內容組合而成。

<h1>
開始標籤
我是標題
內容
</h1>
結束標籤

2-1 範例

HTML 定義了許多元素來顯示不同的內容，可以到W3School官方文件查找。

範例使用了標題 1 與標題 2 用於標示不同層級的文字。

<h1> 我是標題一 </h1>
我是標題一
<h2> 我是標題二 </h2>
我是標題二

3. 屬性

元素可以使用【屬性】，提供了有關元素更多的訊息，屬性必須寫在開始標籤內。

align
屬性名稱
=
指定為
"center"
屬性值

3-1 範例

每個元素有對應的【屬性】可使用，可以到W3School官方文件查找。

範例使用了標題的屬性：對齊，用於控制標題靠齊位置。

<h1 align="center" > 我是標題一 </h1>
我是標題一
<h2 align="right" > 我是標題二 </h2>
我是標題二

3-2 多個屬性

一個元素可以同時使用多個屬性。

範例使用的是連結元素，同時用到兩個屬性。

<a href="http://tw.yahoo.com" target="_blank" > Yahoo奇摩 /<a>
Yahoo奇摩

【href="http://tw.yahoo.com"】告訴瀏覽器要開啟的連結網址。

【target="_blank"】告訴瀏覽器使用新的頁面開啟指定的網址。
