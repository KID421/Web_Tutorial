網頁設計 - CSS 入門

1. 認識 HTML 樣式

【樣式】是 HTML 的屬性，使用CSS語言來調整元素的外觀。

style
樣式
=
指定為
"color : gray;"
"屬性 : 值;"

2-1 範例

CSS 定義了許多屬性來呈現不同的的外觀，可以到W3School官方文件查找。

範例使用顏色屬性將標題改為灰色，此方式稱為行內 CSS。

<h1 style="color:gray;"> 標題 </h1>
標題

2-2 多個屬性

樣式內可以定義多個屬性，記得使用分號區隔開。

<h1 style="color:gray; font-size: 20px;"> 標題 </h1>
標題

3. 外部 CSS

在每個元素內加上樣式修改外觀雖然非常方便，但如果需要同時修改數量龐大的元素可是相對地耗時並且難以維護，考慮到維護性因此出現了外部 CSS的寫法。

3-1 範例

範例告訴瀏覽器要搜尋所有的 h1 並且利用顏色以及字體大小屬性將元素內容文字改為灰色並縮小字體大小。
