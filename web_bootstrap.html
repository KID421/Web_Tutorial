網頁設計 - Bootstrap

一、認識 Bootstrap

Bootsteap 是前端 HTML、CSS、Javascript 的框架，提供樣式、幻燈片、按鈕以及各種網頁元件，讓開發更加容易快速。

Bootstrap 官方文件

二、安裝 Boostrap

如同其他工具一樣，Bootstrap 也提供 CDN，可以前往官方連結複製貼上。

Bootstrap 開始樣板

三、使用 Bootstrap

3-1 網格系統

讓網頁內容固定尺寸使用container

讓網頁內容滿版使用container-fluid

每一列使用row

每一欄使用col注意，每一列裡面的欄位數量總合預設為12

最小尺寸為col-欄位數量

手機尺寸為col-sm-欄位數量

平板尺寸為col-md-欄位數量

桌機尺寸為col-lg-欄位數量

最大尺寸為col-xl-欄位數量

3-2 圖片

讓圖片自動縮放img-fluid

假圖產生器

3-3 麵包屑

麵包屑 breadcrumbs 正確名稱叫做頁面路徑，作用是讓使用者確定目前頁面位置。

3-4 輪播

網頁上常見的輪播在 Bootstrap 裡面的名稱為Carousel

3-5 導覽列

Bootstrap 提供很完整的導覽列功能，包含 RWD 以及漢堡按鈕。

練習

Bootstrap 提供了大量現有的框架，練習使用這些框架設計一個網頁。
